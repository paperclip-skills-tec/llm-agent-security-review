# llm-agent-security-review

Structured security review checklist for LLM-adjacent code. Use this skill whenever you are reviewing code that builds prompts from user input or external data sources, calls any LLM or embedding API, implements RAG retrieval, vector search, or document injection, defines agent tools or plugins, streams or returns LLM output to a client, or configures model endpoints or API keys. LLM-specific threats (prompt injection, insecure output handling, excessive agency, context window data exposure) are a distinct threat class from OWASP Top 10. Invoke this skill even when the AI feature looks simple.
